# 0. ブランドの核(最初に読む)

あなたの条件と強みを重ねると、狙うべきジャンルは一択です。

**「シネマティック暮らしVlog(faceless kurashi vlog)」× フィルムルック × コーヒー × 育児 × ガジェット**

理由:

- 顔・声なしで成立する数少ない「伸びる」ジャンル(手元・後ろ姿・生活音だけで世界観が作れる)
- 日本発でも海外視聴者が5〜7割取れる(字幕は日英併記で世界市場に出せる)
- あなたは既にフィルムエミュレーション(Kodak/Fuji)のプリセットを自作できる = **色で差別化できる数少ない発信者**
- 中庭のある平屋的な注文住宅・新生児・エスプレッソ・ガジェットという「絵になる素材」が家の中に全部ある
- 最終的にLUT・プリセット・アプリ(Totte等)という**自社商品への導線が既に存在する**

一言でいうと:

> **「撮って出しの日常を、フィルムの色で映画にする人」**

# 1. ブランド設計

## 1-1. 伸びる世界観の分析

暮らし系Vlogで登録者10万人超えの共通点:

1. **色が統一されている**(サムネを並べた時に1つの作品に見える)
2. **音が良い**(ASMR的生活音 or 統一されたBGMトーン)
3. **顔が出ないことが「ミステリアスな魅力」に転化している**
4. **1本の中に「小さな物語」がある**(朝→仕込み→完成、のような起承転結)
5. **繰り返し見たくなる「儀式」がある**(毎回同じカップ、同じ窓、同じ猫…)

## 1-2. 競合分析と差別化

| 競合タイプ                     | 強み                     | 弱点(=あなたの入口)                  |
| ------------------------------ | ------------------------ | ------------------------------------ |
| 王道暮らしVlog(女性・料理中心) | 温かい世界観、固定ファン | 色味が量産型(明るいベージュ系ばかり) |
| ミニマリスト系男性Vlog         | ガジェット親和性         | 生活感が薄く、育児層に届かない       |
| コーヒー専門チャンネル         | 深い専門性               | 生活文脈がなく単調、伸び悩みやすい   |
| 海外cinematic vlog             | 圧倒的映像美             | 日本の暮らし・和の光・畳や中庭がない |

**あなたの差別化(3本柱)**

1. **色**: 自作フィルムエミュレーション(Portra/PRO 400H/Gold系)をLUT化し、全動画に適用。「この色=このチャンネル」を作る
2. **家**: 中庭(コの字)の光、太陽光のある暮らし、注文住宅という舞台装置。定点で季節を撮り続けられる
3. **父親×新生児×手元だけ**: 男性の育児Vlogは供給が極端に少なく、顔なしなら競合ほぼゼロ

## 1-3. コンセプト

**コンセプト文(内部用)**

> 中庭のある家で、赤ちゃんと、コーヒーと、カメラと暮らす。
> 顔は映さない。声も出さない。
> それでも伝わる「豊かさ」を、フィルムの色で残す。

**チャンネル説明文(公開用・案)**

> 岡山の小さな家で、家族と静かに暮らしています。
> コーヒーを淹れる音、赤ちゃんの寝息、雨の中庭。
> フィルムのような色で、なんでもない毎日を記録しています。
> A quiet life in Japan — coffee, baby, film-like colors.

## 1-4. ブランド名候補

| 候補                                    | 意図                                           |
| --------------------------------------- | ---------------------------------------------- |
| **中庭とコーヒー / Courtyard & Coffee** | 舞台と儀式をそのまま名前に。検索にも強い       |

推奨は「日本語+英語サブタイトル」構成(例: 中庭とコーヒー | filmlike days)。日本と海外の両取りができます。

## 1-5. ブランドカラー

## 1-6. ターゲット

**メインペルソナ(国内)**

- 28〜40歳、男女半々
- 子育て中 or これから、家づくり・暮らしを整えることに関心
- コーヒー/カメラ/無印・北欧系が好き
- 「疲れて帰ってきて、寝る前に静かな動画を観たい」層

**サブペルソナ(海外)**

- 25〜45歳、日本の暮らし・Japanese aestheticに憧れる層
- "japan slow living" "japanese house tour" 等で流入

## 1-7. 肩書き・キャッチコピー

**肩書き**: 暮らしを現像する人 / Film-look life recorder

**キャッチコピー候補**

- 「なんでもない日を、映画にする。」
- 「顔は映らない。豊かさは映る。」
- 「今日も、いい光でした。」(← 動画の締め定型文としても使用)

## 1-8. 投稿トーン(文章ルール)

- 一人称は「私」or 主語なし。敬語すぎず、詩的すぎず
- 1投稿の文字数: Instagram本文は3〜5行+英語1行
- 絵文字は使わない or ☕️🌿のみ(世界観維持)
- 赤ちゃんは「むすめ/むすこ」等の呼称で統一、顔・名前・泣き声の長回しはNG
- ネガティブ・愚痴・時事ネタは書かない(疲れた人の避難所であるため)

# 2. コンテンツ設計

## 2-1. YouTube(世界観の母艦)

### ロング動画(8〜15分)— 週1本

| シリーズ名               | 内容                                               | 狙い                            |
| ------------------------ | -------------------------------------------------- | ------------------------------- |
| **中庭のある暮らし**     | 1日のルーティンVlog(朝コーヒー→育児→家事→夕方の光) | メイン。季節×天気で無限に作れる |
| **土曜のエスプレッソ**   | 豆開封→グラインド→抽出→ラテアートをASMRで          | コーヒー層獲得。案件の受け皿    |
| **買ってよかったもの**   | 育児/コーヒー/カメラのガジェット紹介(手元のみ)     | アフィリエイトの主戦場          |
| **家、こう作りました**   | 中庭・太陽光・蓄電池・除湿など住宅設備の実録       | 高単価案件(住宅・家電)への布石  |
| **フィルムのように撮る** | iPhone/α7Cでの撮り方+色作り解説(字幕のみ)          | LUT/プリセット販売への導線      |

### ショート(60秒以内)— 週3〜5本

- ロングの「最高の30秒」切り出し(抽出の瞬間/雨の中庭/朝の光)
- 「音だけで伝わるシリーズ」: 豆を挽く音、雨音、ミルクを作る音
- Before/After: 撮って出し→LUT適用の色変化(これは伸びやすい鉄板)

## 2-2. Instagram(ブランドの顔・案件獲得窓口)

- **リール(週4〜5)**: YouTubeショートと同素材。ただし冒頭0.5秒を静止画的な「決めカット」に差し替え
- **フィード(週2)**: 写真3〜10枚カルーセル。「今週の光」「むすめの手と足だけの成長記録」「今日の一杯」
- **ストーリーズ(毎日)**: 撮影裏側、アンケート(「次どっち見たい?」)、新作告知
- **ハイライト**: 機材 / コーヒー / 家 / LUT / About の5つ固定
- グリッドは3列×色の流れを設計(クリーム→グリーン→オレンジのグラデーション配置)

## 2-3. TikTok(発見されるための拡声器)

- 週5〜7本。リールと同素材でOKだが、**テキストフックを1行目に必ず入れる**
  - 例: 「赤ちゃんが寝た後の、35分間。」「このコーヒー、音だけで飲みたくなるはず」
- 伸びやすい型:
  1. 生活音ASMR(冒頭に「イヤホン推奨」)
  2. LUT Before/After(映像系は保存されやすい)
  3. 「顔出しなしでVlogを始める方法」系ノウハウ(字幕のみ)
- コメント返信は「動画で返信」機能を積極活用(ネタが増える)

## 3-4. ネタ切れしない設計

ネタは「掛け算表」で機械的に生成します。

**[場所] × [行為] × [時間/天気] = 企画**

- 場所: 中庭/キッチン/リビング/寝室/車/近所
- 行為: コーヒー/離乳食準備/掃除/写真現像/ガジェット開封/庭いじり
- 時間・天気: 朝5時/夜泣き後/雨/雪/真夏日/停電(蓄電池ネタ)

6×6×6=216通り。季節と成長(赤ちゃんは毎月変わる)を掛けると実質無限です。

# 4. 撮影設計

## 4-1. 機材の使い分け

| 機材                                  | 役割                                           | 比率 |
| ------------------------------------- | ---------------------------------------------- | ---- |
| **iPhone 16 Pro + Blackmagic Camera** | 日常の90%。固定・手持ち・俯瞰すべて            | 70%  |
| **α7C**                               | 「決めカット」専用(ボケ・夕方の低照度・物撮り) | 25%  |
| **iPhone標準カメラ**                  | ストーリーズ用の即撮り                         | 5%   |

## 4-2. Blackmagic Camera App 設定(保存推奨)

- 解像度: 4K 23.98fps(映画感の基本)/ スローは4K 60fps
- コーデック: Apple ProRes不要。HEVC + **Apple Log**(容量と画質のバランス)
- シャッター: 1/50固定(180°ルール)
- ISO: 上限800、基本は室内400
- WB: **固定**(オートWB禁止。朝5000K/昼5600K/夕方4300K目安)
- レンズ: 24mm(1x)基本、48mm(2x)で手元、13mm(0.5x)は中庭の引きのみ
- NDフィルター: 屋外・中庭用に可変ND(iPhone用マグネット式)を1枚追加購入推奨

## 4-3. α7Cの使い所と設定

- ピクチャープロファイル: S-Log3 / S-Gamut3.Cine(色を後で自作LUTで統一するため)
- 4K 24p、SS 1/50、F2.8以下で「赤ちゃんの手×前ボケ」「湯気×逆光」
- レンズが単焦点1本なら35mm F1.8が最優先候補(手元・食卓・中庭すべて対応)

## 4-4. 構図の型(この6つだけ回せばVlogは成立)

1. **定点マスター**: 三脚でキッチン全景(毎回同じ位置=チャンネルの顔)
2. **真俯瞰**: ドリップ・離乳食・開封(俯瞰アームか棚固定)
3. **手元アップ48mm**: 注ぐ・混ぜる・ボタンを押す
4. **窓・中庭越し**: 手前に窓枠やグリーンを入れた「額縁構図」
5. **床レベル**: 赤ちゃんの足・ハイハイ目線(顔は画角外 or 後頭部のみ)
6. **歩きインサート**: 廊下を進む主観(ジンバル不要、脇を締めて0.5xで)

## 4-5. 音の設計

- 生活音が主役。**iPhone外部マイク(例: DJI Mic miniクラス)を1つ導入**すると品質が跳ねます
- 録り分け: 注ぐ音・スイッチ音・雨音は「音だけ別撮り」して編集で重ねる
- BGMは Artlist / Epidemic Sound(商用ライセンス必須。フリー音源の規約違反はチャンネルの死因になります)
- BGM音量: 生活音より-10〜-15dB下げる(生活音:BGM = 7:3)

# 5. 編集設計(CapCutのみで完結)

## 5-1. マスターテンプレート(1度作れば毎回45分編集)

構成(ロング10分の場合):

1. **0:00-0:15 コールドオープン**: その日いちばん良いカット+タイトル字幕(「雨の日の、長い午後。」)
2. **0:15-0:30 定点マスターで「始まり」を宣言**(毎回同じカット=儀式)
3. 本編: 1シーン45〜90秒、シーン間は**カット繋ぎ基本**(トランジション多用は素人感が出る)
4. 中盤に1回「山」(ラテアート完成/赤ちゃんが笑う声/夕立)
5. ラスト: 夕方の光+締め字幕「今日も、いい光でした。」+次回予告1カット

## 5-2. 色(最重要・あなたの武器)

- 自作のフィルムエミュレーション(Portra/PRO 400H系)を**CapCut用LUT(.cube)に変換**して全動画に適用
  - Lightroomプリセット→ .cube化はPythonで自動変換可能(既存のXMPジェネレータースクリプトを流用して私が変換スクリプトを作れます)
- 手順: Apple Log素材 → CapCutで「Apple Log→Rec.709変換LUT」→ 自作フィルムLUT → 微調整(露出のみ)
- **全動画・全SNSで同じLUT**。これがブランドです

## 5-3. 字幕デザイン(声の代わり)

- フォント: 源ノ明朝(Noto Serif JP)Light / 英語はCormorant系
- 色: #EFE6D8(ブランドクリーム)+ 不透明度90%、影なし・縁なし
- 位置: 下から15%、1行12文字以内、表示3〜4秒
- 語り口: 「〜です」ではなく体言止めと余白(例: 「豆は、深煎り。」「まだ、寝てる。」)
- 日英併記: 日本語の下に小さく英語1行(海外リーチが2倍変わります)

## 5-4. SE・テンポ

- SEは原則使わない(生活音が主役)。使うのはショートの「Before/After切替音」のみ
- テンポ: ロングは平均カット4〜6秒(眠れる速度)、ショートは1〜2秒(離脱させない速度)
- ショートの鉄則: **冒頭0.8秒に最強カット**、テキストフック、ループ構造(最後のカットが冒頭に繋がる)

## 5-5. 最後まで見られる仕掛け

- 冒頭で「完成形」を一瞬見せる(オープンループ)
- 字幕で小さな謎を置く(「今日は、失敗した。」→ 何を?)
- 進行を可視化(「朝」「昼」「夕」の章タイトル)
- 終盤に最も美しいカットを配置(視聴維持率グラフの右端を持ち上げる)
